# sevives-labs

The **SevivɘƧ** agency "labs" repo — an open space where we publicly share
snippets, prompts, tools, and experiments.

The goal isn't to ship a finished product, but to document how we work:
what works, what doesn't, and why.

## What's inside

| Folder | Contents |
|---|---|
| `snippets/` | Reusable cheatsheets and code, distilled from real projects. |
| `_bad-examples/` | **Anti-patterns** — code kept as a counter-example, NOT to reproduce (see below). |

> More thematic folders (prompts, tools…) will be added as we have real, useful
> content for them. No empty "just in case" folders.

## ⚠️ `_bad-examples/`

This folder deliberately holds examples of **what NOT to do**. We keep them for
teaching reference, to explain our current architecture choices by contrast.

- **`old-vite-site/`** — an old version of the showcase site in Vite + React.
  Kept to illustrate an approach we've since abandoned. The official site lives
  in another repo and has nothing to do with this code.
